# CH00의 WoW 애드온 모음

CH00이 제작한 WoW 애드온들을 통합 관리하는 레포지토리입니다.

## 애드온 목록

### SimpleWhisper (귓속말)
간단한 WoW 귓속말(Whisper) 메신저 애드온

![Interface](https://img.shields.io/badge/Interface-Classic~Retail-blue)
![Version](https://img.shields.io/badge/Version-1.5.1-green)

- **귓속말 전용 메신저 UI** — 대화 상대 목록 + 채팅 창 분리 레이아웃
- **BNet(배틀넷) 귓속말 지원** — 일반 귓속말과 BNet 귓속말 통합 관리
- **수신 알림** — 소리 알림 (3종 선택), 자동 창 열기
- **전투 모드** — 전투 중 귓속말 처리 방식 선택 (즉시 / 종료후 / 안열기)
- **채팅창 연동** — 채팅창의 이름 클릭 시 SimpleWhisper로 대화 열기
- **/who 자동 조회** — 대화 상대 선택 시 레벨, 직업, 길드 정보 표시
- **미니맵 버튼 + LDB 지원**

### SimpleRepu (평판 가이드)
TBC 평판과 관련 던전을 표시하는 간단한 가이드 애드온

![Interface](https://img.shields.io/badge/Interface-BCC-blue)
![Version](https://img.shields.io/badge/Version-1.0.0-green)

- **평판 현황 표시** — 미니맵/LDB 툴팁으로 TBC 평판 진행도 확인
- **던전 매핑** — 각 평판에 연관된 던전 표시
- **진영별 필터** — 얼라이언스/호드별 해당 평판만 표시

## 설치

1. [Releases](https://github.com/cdcdcd050/simplewhisper/releases)에서 원하는 애드온의 최신 zip 다운로드
2. 압축 해제하여 애드온 폴더를 WoW 애드온 디렉토리에 복사
   - `World of Warcraft/_버전_/Interface/AddOns/`
3. 게임 재시작 또는 `/reload`
